==========Frontend Setup==========


    Navigate to Frontend Directory 

 1- Run "pnpm i" 
 2- Run "pnpm run dev" to start the project
 

==========Backend Setup==========



    Navigate to Backend Directory 

 1- Run "pnpm i" 
 2- Run "pnpm dev" to start the project , it will automatically run the migrations and seeders only for the first time
 



    Note: Make sure to have installed node version v23.11.0 or above , pnpm and postgresql 16 or above
